Python GUI Boilerplate (Tkinter)

Tkinter is a pretty helpful and easy-to-learn GUI (Graphical User Interface), and it can be used for a lot of purposes. framework

                
    # Make sure you make tkinter installed for this project :D
    import tkinter as tk
    from tkinter import ttk
    root = tk.Tk() # Initializes the window
    root.geometry("300x300") # Dimensions of window
    root.title("Hello") # Sets the title of the GUI

    def increment(): # Command for button
        global counter
        label.config(text=f'Counter is [{counter}]!') # fstrings; very useful :D
        counter += 1

    counter = 0
    label = ttk.Label(root, text="Hello World!") 
    label.pack() # Packs it into the screen
    button = ttk.Button(root, text="Increment counter", command=increment) # pass the function as a command
    button.pack() # Packs it into the screen
    root.mainloop() # Starts the main loop; begins displaying stuff

Python Game Boilerplate (Pygame)

 
                
	import pygame
	from pygame.locals import *
	import random
	
	
	class Game:
	    WIDTH, HEIGHT = 500, 500
	    FPS = 24
	    SPEED = 3
	    TITLE = "Template"
	    BLACK = (0, 0, 0)
	    WHITE = (255, 255, 255)
	    RED = (255, 0, 0)
	    GREEN = (0, 255, 0)
	    BLUE = (0, 0, 255)
	    LIGHTBLUE = (0, 128, 255)
	    LIGHTGREEN = (128, 255, 0)
	    YELLOW = (255, 255, 0)
	    ORANGE = (255, 127, 0)
	    CYAN = (0, 255, 255)
	    PURPLE = (128, 0, 255)
	    MAGENTA = (255, 0, 255)
	
	    def __init__(self):
	        pygame.init()
	
	        self.screen = pygame.display.set_mode((self.WIDTH, self.HEIGHT))
	        self.clock = pygame.time.Clock()
	
	        pygame.display.set_caption(self.TITLE)
	
	        self.running = True
	
	        pygame.key.set_repeat(2, 20)
	
	    def randPosition(self):
	        return random.randint(0, self.WIDTH), random.randint(0, self.HEIGHT)
	
	    def tick(self):
	        self.clock.tick(self.FPS)
	
	    def checkQuit(self):
	        for event in pygame.event.get():
	            if event.type == KEYDOWN:
	                if event.key == K_ESCAPE:
	                    self.running = False
	
	            elif event.type == QUIT:
	                self.running = False
	
	    def keyControls(self):
	        for event in pygame.event.get():
	            if event.type == KEYDOWN:
	                if event.key == K_UP:
	                    pass
	                if event.key == K_DOWN:
	                    pass
	                if event.key == K_LEFT:
	                    pass
	                if event.key == K_RIGHT:
	                    pass
	
	    def mainLoop(self):
	        while self.running:
	            self.screen.fill(self.WHITE)
	            self.tick()
	            self.checkQuit()
	            self.keyControls()
	
	            pygame.display.flip()
	
	        pygame.quit()
	
	if __name__ == "__main__":
	    game = Game()
	    game.mainLoop()

Python Tricks

1. F-strings

F-strings are very useful and can save you a lot of time. As an example, let's say you are given 3 numbers and you have to print the three numbers and the sum of them. You can write it as this:


    a = int(input("Number 1: "))
    b = int(input("Number 2: "))
    c = int(input("Number 3: "))
    print("Number 1:", a, " Number 2:", b, " Number 3:", c, " Sum of everything:", a + b + c)

That takes so long to do! We can simplify this by using f-strings. In this case, the print statement would instead become


    print(f"Number 1: {a}, Number 2: {b}, Number 3: {c}, Sum of everything: {a + b + c}.")

In addition to being more handy to code, it is also much more readable! Python basically evaluates the items in { } first and then puts it into the print statement.

2. Lambdas

Let's say you have a function, 

addOne
that takes a number as an argument and returns the number + 1. We can implement the function in python like this:


    def addOne(number):
        return number + 1

Did you know we can simplify this even more? We can use something called lambdas. These are basically functions in shorthand, which can save a lot of time. For example, we can change addOne into


    addOne = lambda (x): x + 1

Lambdas are not a defined function by itself, so it needs a variable, addOne, to define the function. After the "lambda" keyword, you have to define the arguments. In this case, the argument is x. After that, there is a return expression, which automatically returns the expression, in this case, x+1. We can also do this with the addition of three numbers, like this:


    threeNumberAddition = lambda (a, b, c): a + b + c

Here are some examples of lambdas:


    factorial = lambda x: factorial(x-1) * x if x != 0 else 1 # A nice recursive factorial function.
    factorial(51)

    sampleList = [23, 522, 14, 52, 1]
